二：linux(CentOS)环境下java调用C++代码

2.1 编译环境

a.安装gcc和g++

yum install gcc-c++


b.安装jdk
去官网上下载jdk安装包，建议使用rpm安装包，会自动配置环境变量。安装完后如下：

本机的安装目录为：/usr/java/jdk1.8.0_144/，不同版本可能不同。

这里一定要注意不能安装openjdk,因为openjdk没有include目录，编译时需要用到include目录的头文件。

2.2 制作动态库(so库)

a.创建文件夹"jniso",mkdir jniso。
